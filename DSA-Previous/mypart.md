**Sahi formula ye hai:**

```
Important DSA topic (arrays, trees, DP, graphs)
→ Concept samjho
→ 1 Easy + 2-3 Medium solve karo
→ Max 4-5 din
→ Aage badho — wapis aayega revision mein

Basic/foundational topic (loops, conditions)
→ Concept samjho  
→ 3-5 chhote programs likho
→ Max 2 din
→ Aage badho

Hard topic jo samajh nahi aa raha
→ Ek din aur do
→ Phir bhi nahi aaya → skip karo
→ Aage badho → baad mein wapis aao
→ Dusre topics seekhne ke baad 
  khud aasaan lagega
```
